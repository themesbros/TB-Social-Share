# TB-Social-Share

WordPress social share plugin. Fullwidth. Can be placed in a custom position by using function:

`<?php if ( function_exists( 'tb_social_share_display' ) ) { echo tb_social_share_display(); } ?>`

or on top or bottom of the post.

### Supports:
* Facebook
* Twitter
* Google Plus
* Linkedin
* Pinterest
* Reddit
* Tumblr
* Vk
* Email
